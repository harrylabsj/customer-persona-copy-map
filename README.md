# Customer Persona Copy Map

Map customer personas, pain points, objections, benefits, and CTAs into a practical copy matrix.

## Features

- Persona cards with primary needs, barriers, emotional drivers, trust requirements
- Pain-benefit-message matrix: cross-reference personas with copy strategies
- Persona-specific copy snippets: headlines, benefits, objection handlers, CTAs
- Channel recommendations for each persona-message pair
- Evidence strength scoring — clearly separates data-backed from assumed personas
- Assumption auditor identifies which personas need testing before heavy investment
- Message adapter rewrites core messaging for different audience segments

## Install

```
openclaw skills install harrylabsj/customer-persona-copy-map
```

## Usage

```
Map copy to our 3 customer personas — skincare brand
Create persona-based messaging for a kitchen gadget
Adapt our core product message for different buyer types
Audit our personas — which ones are backed by data?
Write persona-specific ad copy for each audience segment
```

## Platforms

Product pages, ads, email, landing pages, social media

## Safety

Assumptions clearly labeled. No stereotyping of protected traits. No exploitative messaging targeting vulnerabilities. Evidence basis always stated.
